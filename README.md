# 🌐 Tira Beauty Website Clone (Frontend Project)

## 📌 Project Overview
This project is a **frontend clone of the Tira Beauty website**, developed using **HTML5 and CSS3**. It focuses on building a visually appealing user interface with structured layout and modern design techniques.

## 🚀 Features
- **Product listing cards** with images, price, ratings, and discounts
- **Login/Sign Up page navigation**
- Integrated **videos and banner sections**
- Styled **Add to Cart button** with icons
- Clean **UI design with typography and color styling**

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- Flexbox & Grid  
- Font Awesome (Icons)

## 🎯 Key Highlights
- Built using **semantic HTML structure**
- Applied **CSS Box Model, positioning, and layout techniques**
- Focused on **user interface (UI) and user experience (UX) design**
- Hands-on practice in **frontend development**

## 📂 Project Structure
```
/project-folder
│── index.html
│── style.css
│── images/
│── videos/
```

## 🔗 How to Run
1. Download or clone the repository  
2. Open `index.html` in your browser
